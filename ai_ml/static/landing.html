Models in Production

📊
● LIVE

Habit Scoring Model

Weighted formula combining wake-up consistency (35%), challenge completion (25%), snooze reduction (20%), and sleep schedule adherence (20%) into a single habit score.

GET /users/{user_id}/habit-score
🎯
● LIVE

Adaptive Difficulty Model

Elo-style reinforcement engine that replays a user's real challenge attempts and returns their current rating plus the next recommended difficulty level.

GET /users/{user_id}/difficulty
🧠
● LIVE

Behavior Analysis Model

Analyzes snooze patterns, wake-up confirmation behavior, and sleep-duration adherence to produce the four normalized inputs the Habit Scoring Model needs.

GET internal · feeds habit-score
💡
● LIVE

Recommendation Model

Identifies each user's weakest habit-score component and maps it to a targeted, content-library recommendation for sleep, wake-up, or habit improvement.

GET /users/{user_id}/recommendation
⚠️
● LIVE

Prediction Model

XGBoost classifier trained to flag oversleep / snooze risk, achieving 92.5% validation accuracy, serialized as snooze_predictor.joblib.

GET /users/{user_id}/risk-prediction
