рџ“± Project Overview

The Intelligent Cognitive Alarm Platform (ICAP) is an AI-powered mobile application designed to improve users' wake-up habits through cognitive engagement. Unlike traditional alarm applications, users cannot dismiss alarms directly. Instead, they must complete AI-generated cognitive challenges. The platform provides role-based access for Users, Wellness Coaches, and Administrators, enabling performance monitoring, Habit Score analytics, and user management.

Implements secure role-based authentication and authorization.
