🧠 Intelligent Cognitive Alarm Platform (ICAP)

An AI-powered mobile alarm application that encourages healthy wake-up habits by requiring users to solve cognitive challenges before dismissing alarms. The platform also provides role-based access for Users, Wellness Coaches, and Administrators to monitor productivity and sleep habits.

---

📌 Project Overview

Traditional alarm applications allow users to dismiss alarms instantly, often leading to oversleeping. ICAP addresses this problem by introducing AI-generated cognitive challenges that users must complete before the alarm stops.

The application is designed with a secure, scalable full-stack architecture using React Native and FastAPI.
