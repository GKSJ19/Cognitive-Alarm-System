Welcome to the **Intelligent Cognitive Alarm Platform (ICAP)** documentation repository for **Milestone 1**.

This project aims to revolutionize morning routines by integrating cognitive puzzles with alarms, ensuring users are mentally engaged when they wake up. Milestone 1 establishes the enterprise-grade foundation for the application, featuring a fully functional authentication system, robust database architecture, and a modern, responsive user interface.

## 🚀 Milestone 1 Highlights

- **Secure Authentication:** JWT-based login and registration with bcrypt password hashing.
- **Role-Based Access Control (RBAC):** Tiered permissions for Admins, Users, and Coaches.
- **Interactive Dashboard:** A premium, visually stunning dashboard using Recharts and Framer Motion.
- **Database Architecture:** Fully asynchronous SQLAlchemy 2.x setup with Alembic migrations.
