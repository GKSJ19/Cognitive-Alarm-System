# Intelligent Cognitive Alarm Platform - Mobile App

This React Native / Expo application provides a production-style mobile frontend for the Intelligent Cognitive Alarm Platform.

## Features
- JWT-based authentication flow
- Home dashboard, alarms, analytics, profile, coach, and admin views
- Redux Toolkit state management
- React Hook Form + Zod validation
- REST API integration via Axios

## Setup
1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env` and set your backend URL.
3. Start Expo: `npm start`
