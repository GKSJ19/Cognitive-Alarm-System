# MongoDB Documentation - Intelligent Cognitive Alarm Platform

## Database Name

cognitive_alarm_db

## Purpose

MongoDB is used as the secondary database of the Intelligent Cognitive Alarm Platform.

It stores flexible and semi-structured data required by different modules of the application.

---

## Collections

### 1. challenge_data

**Purpose:**

Stores cognitive challenge information used by the Cognitive Challenge Engine.

### 2. user_preferences

**Purpose:**

Stores user difficulty preferences and challenge preferences.

### 3. logs

**Purpose:**

Stores wake-up verification logs and system activity.

### 4. notifications

**Purpose:**

Stores reminders and notification information.

### 5. content

**Purpose:**

Stores recommendation and reusable content.

---

## Source

This MongoDB design is based on the Intelligent Cognitive Alarm Platform project PDF.
