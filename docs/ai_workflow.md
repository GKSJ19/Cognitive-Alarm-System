done# AI Workflow

```text
Alarm Rings
      │
      ▼
Generate Cognitive Challenge
      │
      ▼
User Attempts Challenge
      │
      ▼
Evaluate Performance
      │
      ▼
Analyze User Behaviour
      │
      ▼
Update Difficulty Level
      │
      ▼
Calculate Habit Score
      │
      ▼
Generate Personalized Recommendation
```

## Objective

The workflow ensures that users are mentally engaged before dismissing an alarm while continuously adapting the experience based on their performance and habits.
