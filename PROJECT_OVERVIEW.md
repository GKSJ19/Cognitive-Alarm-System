# Project Overview

## 🎯 Purpose
The **Intelligent Cognitive Alarm Platform (ICAP)** is designed to rethink how individuals wake up and start their day. By requiring users to solve cognitive puzzles (math, memory, logic) to dismiss their alarms, the platform ensures users are fully awake and mentally engaged.

## 🏢 Target Audience
- **Individuals** seeking to improve their morning routines and eliminate oversleeping.
- **Wellness Coaches** monitoring client adherence to sleep and habit goals.
- **Data Enthusiasts** looking to track their cognitive performance and habit scores over time.

## 📈 Scope (Milestone 1)
This project is being developed in iterative milestones. The current iteration (**Milestone 1**) is focused purely on laying a robust, enterprise-grade foundation. This includes the boilerplate, structural design, UI/UX framework, database schema, and security layers. 

### Milestone 1 Objectives
1. **System Foundation:** Establish a secure, scalable Client-Server architecture.
2. **Frontend UI/UX:** Implement a stunning, responsive, dark-mode user interface with interactive elements.
3. **Backend API:** Build an asynchronous RESTful API capable of robust data validation.
4. **Data Persistence:** Design and implement a relational database schema.
5. **Security:** Secure the platform with modern authentication and authorization strategies.

## 🔮 Future Scope
Future milestones will introduce:
- Complex AI-driven cognitive puzzle generation.
- Dynamic alarm scheduling and progressive difficulty adjustments.
- Advanced analytics and wellness coaching dashboards.
- External integrations (wearables, sleep trackers).
*(Note: These features are not implemented in Milestone 1).*
